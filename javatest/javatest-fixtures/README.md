# Fixtures

_______

You can include this module with this dependency: 

```xml
<dependency>
    <groupId>org.javatest</groupId>
    <artifactId>javatest-fixtures</artifactId>
    <version>${javatest.version}</version>
    <scope>test</scope>
</dependency>
```
